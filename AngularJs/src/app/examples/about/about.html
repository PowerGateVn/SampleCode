Sails.js + AngularJS Boilerplate

This is an example boilerplate for your Sails.js + AngularJS combo project. You can use this boilerplate as a starting point of your own awesome modern web application. In addition there are many example to show you how to get started.

Main goals of this boilerplate

  • Totally separated back- and frontend development
  • Really light frontend application
  • Provide an example application of sails.js + angular to community
  • Secure authentication with backend
  • Power of AngularJS + WebSockets
  • Simple examples of how to use these

Below is detailed information about this boilerplate.

"Sides" of this boilerplate

  • Backend
    • Only REST API, no frontend stuff - really just an API, nothing else.
    • Passport.js for user authentication. Supported auth types: local, twitter, facebook, google+, etc.
    • JWT support for API calls. JSON Web Tokens are used to to authenticate the user after a successfull login.
    • CORS support. Really that is what we need to do when separating back- and frontend.
    • Real business logic and validations for data Backend handles data related validations, logging, and user authentication.
  • Frontend
    • A single-page application (also known as single-page interface or SPI) is a web application or web site that fits on a single web page with the goal of providing a more fluid user experience akin to a desktop application.
    • Lightweight only client side application
    • Authentication to backend
    • Route examples for views
    • Support for WebSocket communication with backend
    • Interceptor for HTTP / Socket communication

Libraries used and other stuff

  • Backend

    • Passport Passport is authentication middleware for Node.js. Extremely flexible and modular, Passport can be unobtrusively dropped in to any Express-based web application. A comprehensive set of strategies support authentication using a username and password, Facebook, Twitter, and more.
      • Local
      • Twitter
      • GitHub
      • Facebook
      • Google
    • JsonWebToken JSON Web Token implementation (symmetric and asymmetric)
    • barrels Simple DB Fixtures for Sails.js (^0.10.x)
    • ua-parser A multi-language port of Browserscope's user agent parser.
  • Frontend
    • slush-angular A slush generator for AngularJS using the Google Angular App Structure Recommendations
    • jQuery jQuery is a fast, small, and feature-rich JavaScript library.
    • Bootstrap Bootstrap is the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.
    • Font Awesome Font Awesome gives you scalable vector icons that can instantly be customized — size, color, drop shadow, and anything that can be done with the power of CSS.
    • AngularJS AngularJS lets you write client-side web applications as if you had a smarter browser. It lets you use good old HTML (or HAML, Jade and friends!) as your template language and lets you extend HTML’s syntax to express your application’s components clearly and succinctly.
    • AngularUI Router The de-facto solution to flexible routing with nested views
    • UI Bootstrap Bootstrap components written in pure AngularJS by the AngularUI Team
    • angular-moment Moment.JS directives for Angular.JS (timeago and more)
    • angular-bootstrap-show-errors An Angular directive for Bootstrap to intelligently show form validation errors
    • angular-linkify Angular filter, directive, and service to linkify text.
    • angular-toastr Angular port of CodeSeven/toastr.
    • textAngular A radically powerful Text-Editor/Wysiwyg editor for Angular.js.
    • sails.io.js Browser SDK for communicating w/ Sails via sockets
    • angularSails AngularJS bindings for Sails.
  • Common stuff Libraries that are used on both sides
    • Lo-Dash A utility library delivering consistency, customization, performance, & extras.
    • Moment.js + Moment Timezone Parse, validate, manipulate, and display dates in javascript.
